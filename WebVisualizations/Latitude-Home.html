Summary: Latitude vs. X

The purpose of this project was to analyze weather information based on a cityâ€™s position relative to the equator. We used OpenWeatherMap API to generate a dataset of over 500 cities randomly located across the globe, including weather variables of maximum temperature, humidity, cloudiness, and wind speed.

We used Matplotlib to plot the results of weather variables vs. latitude. This web site provides the source data, as well as individual visualizations and comparisons created from this analysis. Descriptions of the data and trends observed are also included.
